# 浮动

元素的水平方向浮动，意味着元素只能左右移动而不能上下移动。

一个浮动元素会尽量向左或向右移动，直到它的外边缘碰到包含框或另一个浮动框的边框为止。

浮动元素之后的元素将围绕它。

浮动元素之前的元素将不会受到影响。
当把几个浮动的元素放到一起，如果有空间的话，它们将彼此相邻。

## 设置浮动

值：

- left
- right
- none
- inherit

## 清楚浮动

元素浮动之后，周围的元素会重新排列，为了避免这种情况，使用 clear 属性。
clear 属性指定元素两侧不能出现浮动元素。

值：

- left
- right
- both
- none
- inherit

```css
.text_line
{
    clear:both;
}
```
