## 2、设计游戏主循环

### 逻辑实现

~~~
Tetris.cpp
中在play方法中调用另一个成员函数init
任何游戏都是一个死循环while(1)加入循环

常用步骤：
1、接受用户输入 2、渲染游戏画面(添加定时器、时延，以达到刷新目的，到后面刷新更快)

在.h中创建一个接口方法点击事件，因为这是内部用的接口方法，所以放在private里面比较合适，其余同理


private:
	void keyEvent();

并实现具体方法在cpp中，并且添加时延参数

考虑到后面越来越快 将延时判断不写死 写成数据成员变量
另起一个
private:
 int delay;
并在初始化中先给个初始值

考虑到方块移动的时候如果到时延更新的话玩家有时会感到明显的卡顿，
因此添加一个新的成员变量update来进行操作后实时更新,这样在设置点击事件里让update为真时可立马刷新窗口
 bool update;
 
添加下降方法drop 和 更新游戏行列方法 clearLine

并在时延一更新就调用drop方法， 除了更新游戏窗口外还更新游戏行列
~~~



### 代码实现

~~~c++
#pragma once
class Tetris 
{
public:
	// 构造函数  参数有 行列 左边界 顶部边界 方块大小
	Tetris(int rows,int cols,int left,int top,int blockSize);
	// 初始化
	void init();
	// 开始游戏
	void play();

private:
	void keyEvent();
	void updateWindow();// 更新游戏画面(即渲染)
	int getDelay(); //返回距离上一次调用该函数，间隔了多少事件(ms)
	void drop();
	void clearLine();

private:
	int delay;
	bool update; //是否更新
};
~~~



~~~c++
#include "Tetris.h"

Tetris::Tetris(int rows, int cols, int left, int top, int blockSize) {}

void Tetris::init() 
{
	delay = 30;
}

void Tetris::play() {
	init();

	//定时器
	int timer = 0;
	while (1) {
		// 接受用户输入  因此要创建点击事件
		keyEvent();

		timer += getDelay();
		if (timer > delay)
		{
			timer = 0;
			update = true;
			drop();
		}
		if (update)
		{   
		// 渲染游戏画面
		updateWindow();
		// 更新游戏数据
		clearLine();
		}
	}
}

void Tetris::keyEvent() {}

void Tetris::updateWindow() {}

int Tetris::getDelay() {
	return 0;
}

void Tetris::drop() {}

void Tetris::clearLine() {}
~~~



## 3、创建新方块

### 形态设计

~~~c++
使用一个4*2的二维数组表示各种形态 当然目前还没有考虑到旋转时的形态变化，等后续使用算法来弥补 目前只考虑初始形态设计
    0 1
    2 3

若1357则为长条型I  2457为Z1型  3546为Z2型 3547为T型或山型 
  2357为J型 2345为田型
~~~



### 逻辑实现

~~~c++
可以在block.cpp中定义一个全局数组 或者在block构造函数里定义
    
然后随机生成一种方块 需要导入stdlib库
    调用rand方法 rand的取值范围是0到RAND_MAX 32767
    注意rand()%7范围是0到6  +1才是1至7
    但这种随机并不真随机，需要在之前先执行srand()
    
在tetris.cpp中初始化配置随机种子  导入stdlib库和time库
    调用srand方法（用于初始化随机种子）
    	//配置随机种子 参数直接给定当前时间戳time(NULL)  
	srand(time(NULL));

然后将随机方块作为方块类型 在私有成员变量中创建blockType

考虑到要在游戏里考虑这个方块的坐标 因此需要来用两个变量来记录坐标，因此可以封装成一个结构体
    struct Point{
        int row;
        int col;
    }
然后需要记录四个小方块，因此使用结构体数组 在私有成员变量中创建
    Point smallBlocks[4];

假设将1357放到游戏里初始化 以游戏左上角为00
       它对应的坐标为 01 11 21 31
    发现一个规律就是等于2*行数+列数
    
因此小方块用循环方式放在初始位置




~~~



### 代码实现

~~~c++
#pragma once

struct Point {
	int row;
	int col;
};

class Block 
{
public:
	Block();
	void drop();  //下降
	void moveLeftRight(int offset); //左右移，给参数大于零右移，小于零左移
	void rotate(); //旋转
	void draw(int leftMargin, int topMargin);  //画方块 给定左顶边界


private:
	int blockType; //方块类型
	Point smallBlocks[4];
};


#include "Block.h"
#include <stdlib.h>
Block::Block() 
{
	int blocks[7][4] = {
	1,3,5,7,
	2,4,5,7,
	3,5,4,6,
	3,5,4,7,
	2,3,5,7,
	3,5,7,6,
	2,3,4,5,
	};
	 
	//随机生成一种俄罗斯方块 值为1-7
	blockType = rand() % 7 + 1;

	for (int i = 0; i < 4; i++) {
		int value = blocks[blockType-1][i];
		smallBlocks[i].col = value / 2;
		smallBlocks[i].row = value % 2;
	}


}

void Block::drop() {}

void Block::moveLeftRight(int offset) {}

void Block::rotate() {}

void Block::draw(int leftMargin, int topMargin) {}
~~~

### 代码实现

~~~c++
#pragma once
#include <graphics.h>


struct Point {
	int row;
	int col;
};

class Block 
{
public:
	Block();
	void drop();  //下降
	void moveLeftRight(int offset); //左右移，给参数大于零右移，小于零左移
	void rotate(); //旋转
	void draw(int leftMargin, int topMargin);  //画方块 给定左顶边界


private:
	int blockType; //方块类型
	Point smallBlocks[4];
	IMAGE *img;

	static IMAGE* imgs[7];
	static int size; //表示大小

};


#include "Block.h"
#include <stdlib.h>
IMAGE* Block::imgs[7] = { NULL };
int Block::size = 36;


Block::Block() 
{
	if (imgs[0]==NULL)
	{
		IMAGE imgTmp;
		loadimage(&imgTmp, "res/tiles.png");

	//设置切割对象
		SetWorkingImage(&imgTmp);
		for (int i = 0; i < 7; i++) {
			imgs[i] = new IMAGE;
			getimage(imgs[i], i * size, 0, size, size);//存放位置  切割点x轴坐标 y轴坐标  切割长度 宽度
		}
		SetWorkingImage();

	}

	int blocks[7][4] = {
	1,3,5,7,
	2,4,5,7,
	3,5,4,6,
	3,5,4,7,
	2,3,5,7,
	3,5,7,6,
	2,3,4,5,
	};
	 
	//随机生成一种俄罗斯方块 值为1-7
	blockType = rand() % 7 + 1;

	for (int i = 0; i < 4; i++) {
		int value = blocks[blockType-1][i];
		smallBlocks[i].col = value / 2;
		smallBlocks[i].row = value % 2;
	}

    //配置方块图片
	img = imgs[blockType - 1];


}

void Block::drop() {}

void Block::moveLeftRight(int offset) {}

void Block::rotate() {}

void Block::draw(int leftMargin, int topMargin) {}
~~~
